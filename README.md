# @muath-ye/catalog

A `cat` on **Windows** is a Node.js command-line app that reads the contents of a file or URL and prints them to the console.

## Installation

Install the package globally using the following commad

```sh
npm i -g @muath-ye/catalog
```

## Usage

```sh
cat /path/to/file
```
or

```sh
cat /url/to/file
```
